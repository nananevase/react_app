## Installation

To install and run this project locally, follow these steps:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/nananevase/react_app.git
    cd react_app
    ```

2. **Install the dependencies:**

    ```bash
    yarn install
    ```
